\# StudentsApi



This is an ASP.NET Core Web API for managing students, courses, and enrollments.



\## Features

\- CRUD operations for students and courses

\- Entity relationships with EF Core

\- RESTful endpoints



\## Getting Started

1\. Clone the repo

2\. Run `dotnet build`

3\. Start with `dotnet run`



\## Technologies

\- ASP.NET Core

\- Entity Framework Core

\- SQL Server
